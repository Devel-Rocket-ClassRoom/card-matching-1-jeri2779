# 미니 게임 과제: 카드 짝 맞추기 (Card Matching)

## 과제 개요

콘솔에서 동작하는 **카드 짝 맞추기 게임**을 구현하세요.

배포된 실행 파일(`Card-Matching-1.exe`)을 실행하여 완성된 게임을 확인할 수 있습니다.

---

## 과제 방식: 개인

이 과제는 **개인**으로 수행합니다.

## 게임 규칙

- 4x4 격자(총 16장)에 1~8까지의 숫자가 **2장씩** 뒤집어져 있습니다.
- 게임 시작 시 카드를 무작위로 섞습니다.
- 매 턴마다 **행과 열 번호**를 입력하여 카드 **2장**을 선택하고 뒤집습니다.
- 두 카드의 숫자가 **같으면** 짝이 맞은 것이며, 카드가 공개된 상태로 유지됩니다.
- 두 카드의 숫자가 **다르면** 다시 뒤집힙니다.
- **8쌍을 모두 찾으면** 게임 클리어입니다.

## 추가 도전 (선택)

기본 게임을 완성했다면 다음 기능을 추가하세요.

### 난이도 선택

게임 시작 시 보드 크기를 선택할 수 있습니다.

| 난이도 | 보드 크기 | 카드 쌍 |
|--------|----------|--------|
| 쉬움 | 2x4 (8장) | 4쌍 |
| 보통 | 4x4 (16장) | 8쌍 |
| 어려움 | 4x6 (24장) | 12쌍 |

**실행 예시:**

```
=== 카드 짝 맞추기 게임 ===

난이도를 선택하세요:
1. 쉬움 (2x4)
2. 보통 (4x4)
3. 어려움 (4x6)
선택: 1

카드를 섞는 중...

    1열 2열 3열 4열
1행  **  **  **  **
2행  **  **  **  **

시도 횟수: 0 | 찾은 쌍: 0/4
```

### 시도 횟수 제한

난이도별로 시도 횟수가 제한됩니다. 제한 횟수 내에 모든 쌍을 찾지 못하면 게임 오버입니다.

| 난이도 | 시도 제한 |
|--------|----------|
| 쉬움 | 10회 |
| 보통 | 20회 |
| 어려움 | 30회 |

**실행 예시:**

```
시도 횟수: 19/20 | 찾은 쌍: 6/8

첫 번째 카드를 선택하세요 (행 열): 2 1

...

짝이 맞지 않습니다!

=== 게임 오버! ===
시도 횟수를 모두 사용했습니다.
찾은 쌍: 6/8

새 게임을 하시겠습니까? (Y/N):
```
